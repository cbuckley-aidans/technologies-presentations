Text Encoding
How computers turn letters into binary — and why we needed more than one system to do it.
The Code Book Analogy
Text Encoding
A system that maps every character to a unique binary number. The computer uses this mapping like a reference book — look up the character, get the bits.

Key idea: the encoding system is the agreement between sender and receiver about what the bits mean. Without the same code book, the bits are meaningless.
The First Standard
ASCII — 7 bits per character
American Standard Code for Information Interchange (1963). Assigns a unique 7-bit binary value to 128 characters: letters, digits, punctuation, and control codes.
character
decimal
binary
Why 7 bits?
27 = 128 possible values. Enough for English uppercase, lowercase, digits, and common punctuation. The 8th bit was originally used for error checking (parity).
Stored as 8 bits (1 byte)
In practice each ASCII character is stored in one byte, with a leading 0 padding the 7-bit value. So 'A' (65) becomes 01000001.
Limitation: 128 characters is only enough for English. No accents, no other scripts, no symbols beyond basic punctuation.
Another encoding system
Morse Code — a different code book
Morse code is another text encoding system. Instead of bytes, it uses dots and dashes. Like ASCII, it is a code book: each character maps to a unique signal pattern.

On your whiteboard
Is Morse code a more efficient encoding system than ASCII binary?
Explain your reasoning. Think about what "efficient" might mean.
Consider: symbol count
Common letters like E and T use fewer Morse symbols than 8-bit ASCII. Rare letters use more.
Consider: fixed vs variable
ASCII is always 8 bits per character. Morse length varies. Which is easier for a computer to process?
Consider: what is missing
Morse only covers a subset of characters. ASCII covers 128. Efficiency is not just about bits.

Beyond ASCII
Unicode — one standard for every language
The Problem
ASCII worked for English, but every country invented its own encoding for their script. Files from different systems were often unreadable. The world needed one universal code book.
The Solution: Unicode (1991)
A single standard that assigns a unique code point to every character in every writing system on Earth — over 149,000 characters and counting.
